datalovers
  • DataLovers is a web application that visualizes 100 recent tweets and 100 useful websites for early learners who want to initially explore the field of data visualization. This project was initiated from three goals: 1)Designing an interface that enables data beginners to intuitively explore the world of data vis. 2) Visualizing datavis lovers' locations in the real world 3)Designing a graph optimized in the mobile phone environment.
