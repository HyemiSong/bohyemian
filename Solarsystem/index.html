Solar System
  • Solar System is an audio-visual performance. A goal of this project was devising methods to design a soundtrack using data. Based on ideas of the potential for the creation of a data-driven soundtrack through through the generation of pattern repetition, a soundtrack and a visual performance were created with the orbit data of eight planets in the Solar System.
